 <h1>GuatequeStore</h1>
    <h2>Plataforma de gestión y venta de productos de moda</h2>
    
    <p>El objetivo principal de este proyecto es desarrollar una plataforma completa para la gestión y venta de productos de moda. Esta plataforma simula el funcionamiento de una tienda de ropa real, permitiendo a los usuarios explorar el catálogo, consultar tallas y colores disponibles, gestionar un carrito de compras y completar pedidos.</p>
    
    <p>Además de la experiencia para el cliente, el proyecto contempla herramientas internas que permiten a los administradores mantener actualizado el inventario, gestionar productos, revisar ventas y optimizar la experiencia de compra.</p>
    
    <h2>Moda responsable y economía circular</h2>
    <h3>Transformando la industria de la moda</h3>
    
    <p>Guateque impulsa un modelo más responsable y sostenible que reduce el desperdicio y promueve la economía circular.</p>
    <p>Trabajamos con proveedores que cumplen estándares ESG y fomentamos devoluciones responsables: las prendas devueltas, con recibo, pueden ser recicladas y los clientes obtienen recompensas, cerrando el ciclo de vida del producto.</p>
    
    <h2>Estructura del proyecto</h2>
    
    <p>El repositorio está organizado de la siguiente manera:</p>
    
    <ul>
        <li>Android/ → Aplicación móvil desarrollada en Java</li>
        <li>Frontend/ → Aplicación web desarrollada en React</li>
        <li>Backend/ → API y lógica del servidor desarrollados en Java con Spring Boot</li>
    </ul>
    
    <h3>Frontend</h3>
    <p>Framework: React</p>
    <p>Permite interactuar con el sistema, visualizar productos, proveedores y eventos.</p>
    <p>Se puede incluir una captura de pantalla del frontend para mostrar la interfaz visual.</p>
    
    <h3>Backend</h3>
    <p>Lenguaje y framework: Java con Spring Boot</p>
    <p>Expone APIs REST para gestionar productos, proveedores, pedidos y eventos.</p>
    
    <h3>Android</h3>
    <p>Aplicación móvil desarrollada en Java</p>
    <p>Permite a los usuarios acceder a funcionalidades similares a la web desde dispositivos Android.</p>
    
    <h2>Tecnologías utilizadas</h2>
    
    <ul>
        <li>Frontend: React, JavaScript, HTML, CSS</li>
        <li>Backend: Java, Spring Boot, Maven/Gradle</li>
        <li>Android: Java, Android Studio</li>
        <li>Control de versiones: Git / GitHub</li>
    </ul>
    
    <h2>Cómo ejecutar el proyecto</h2>
    
    <p>Frontend: instalar dependencias y ejecutar la app en un servidor local.</p>
    <p>Backend: correr la aplicación Spring Boot para exponer las APIs.</p>
    <p>Android: abrir el proyecto en Android Studio y ejecutar en un emulador o dispositivo físico.</p>
    
    <h2>Buenas prácticas al integrar módulos</h2>
    
    <p>Trabaja en ramas separadas por módulo y luego integra cada módulo a la carpeta correspondiente (Android/, Frontend/, Backend/).</p>
    <p>Antes de subir cambios, revisar siempre git status y diferencias para evitar sobrescribir trabajo remoto.</p>
