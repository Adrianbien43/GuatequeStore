<h1>GuatequeStore</h1>
<h2>Plataforma de gestión y venta de productos de moda</h2>

<p>El objetivo principal de este proyecto es desarrollar una plataforma completa para la gestión y venta de productos de moda. Esta plataforma simula el funcionamiento de una tienda de ropa real, permitiendo a los usuarios explorar el catálogo, consultar tallas y colores disponibles, gestionar un carrito de compras y completar pedidos.</p>

<p>Además, proporciona herramientas internas que permiten a los administradores mantener actualizado el inventario, gestionar productos, revisar ventas y optimizar la experiencia de compra.</p>

<hr>

<h2>Moda responsable y economía circular</h2>
<h3>Transformando la industria de la moda</h3>

<p>Guateque impulsa un modelo más responsable y sostenible, promoviendo la economía circular y reduciendo el desperdicio. Trabajamos con proveedores que cumplen estándares ESG y fomentamos devoluciones responsables: las prendas devueltas, con recibo, pueden ser recicladas, mientras los clientes obtienen recompensas, cerrando el ciclo de vida del producto.</p>

<hr>

<h2>Estructura del proyecto</h2>

<p>El repositorio está organizado de la siguiente manera:</p>
<ul>
    <li><strong>Android/</strong> → Aplicación móvil desarrollada en Java</li>
    <li><strong>Frontend/</strong> → Aplicación web desarrollada en React</li>
    <li><strong>Backend/</strong> → API y lógica del servidor desarrollados en Java con Spring Boot</li>
</ul>

<h3>Frontend</h3>
<p><strong>Framework:</strong> React</p>
<p>Permite interactuar con el sistema, visualizar productos, proveedores y eventos. Se puede incluir una captura de pantalla del frontend para mostrar la interfaz visual.</p>

<h3>Backend</h3>
<p><strong>Lenguaje y framework:</strong> Java con Spring Boot</p>
<p>Expone APIs REST para gestionar productos, proveedores, pedidos y eventos.</p>

<h3>Android</h3>
<p><strong>Aplicación móvil desarrollada en Java</strong></p>
<p>Permite a los usuarios acceder a funcionalidades similares a la web desde dispositivos Android.</p>

<hr>

<h2>Tecnologías utilizadas</h2>
<ul>
    <li><strong>Frontend:</strong> React, JavaScript, HTML, CSS</li>
    <li><strong>Backend:</strong> Java, Spring Boot, Maven/Gradle</li>
    <li><strong>Android:</strong> Java, Android Studio</li>
    <li><strong>Control de versiones:</strong> Git / GitHub</li>
</ul>

<hr>

<h2>Cómo ejecutar el proyecto</h2>
<ul>
    <li><strong>Frontend:</strong> instalar dependencias y ejecutar la app en un servidor local.</li>
    <li><strong>Backend:</strong> correr la aplicación Spring Boot para exponer las APIs.</li>
    <li><strong>Android:</strong> abrir el proyecto en Android Studio y ejecutar en un emulador o dispositivo físico.</li>
</ul>

<hr>

<h2>Buenas prácticas al integrar módulos</h2>
<p>Trabaja en ramas separadas por módulo y luego integra cada módulo a la carpeta correspondiente (<strong>Android/</strong>, <strong>Frontend/</strong>, <strong>Backend/</strong>). Antes de subir cambios, revisar siempre <code>git status</code> y diferencias para evitar sobrescribir trabajo remoto.</p>
